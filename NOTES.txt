# Idee
- Live Reload von statischen Files via webpack-livereload-plugin
  https://github.com/statianzo/webpack-livereload-plugin
- "Künstliches" einbinden aller HTML-Dateien ins Bundle, um bei Änderungen einen Webpack-Rebuild zu triggern, der wiederum 
- Start in docker-compose per npm run start (Webpack-Deveserver) oder node index.js (Webpack-Devserver per API/Script, für Play.JS auf iPad)
